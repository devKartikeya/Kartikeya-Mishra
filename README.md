🚀 Kartikeya Mishra Portfolio

A premium, animated developer portfolio built with **React**, **GSAP**, and **TailwindCSS**.  
This project showcases my skills, projects, and contact information in a sleek single‑page design with smooth animations and glassmorphism styling.

---

## ✨ Features
- **Hero Section**: Animated introduction with typing effects and bouncing buttons.
- **About Section**: Premium glassmorphism design with responsive layout.
- **Projects Section**: Showcases selected work with animations and hover effects.
- **Skills Section**: Icon‑rich grid with staggered GSAP animations.
- **Contact Section**: Functional email form powered by **EmailJS**.
- **Responsive Navbar**: Smooth scroll navigation with mobile drawer.

---

## 🛠️ Tech Stack
- **React** (Frontend framework)
- **TailwindCSS** (Utility‑first styling)
- **GSAP + ScrollTrigger** (Animations)
- **React Icons** (Skill & social icons)
- **EmailJS (@emailjs/browser)** (Contact form integration)
